# 频道功能 UI/UX 重构方案 / Channel UX Redesign

> **日期**: 2026-07-09 | **对标**: 微信公众号 + 知识星球 | **范围**: Flutter 移动端

## 一、问题诊断

### 现状
- `channel_detail_page.dart` **1659 行**，23 个方法，混合 6 类不相关职责
- 消息项采用"聊天气泡"思维，缺乏内容消费的沉浸感
- 付费流程 390 行逻辑混在详情页，状态不透明

### 四大痛点（用户确认）
1. **频道详情页臃肿** —— 上帝页面，首屏卡顿，状态机混乱
2. **消息流阅读体验差** —— 套了频道皮的聊天框，无社交反馈感
3. **订阅/付费流程混乱** —— 状态不透明，错误恢复散落
4. **创建与编辑体验差** —— 表单验证、分类/封面/价格交互粗糙

## 三、消息卡片新设计（核心体验提升）

### 统一骨架（4 种类型共享）
```
┌─────────────────────────────────────────┐
│ 📌 [置顶]            🕐 2小时前 · 👁 1.2k │  顶栏：状态 + 元信息
├─────────────────────────────────────────┤
│            [内容区：按类型渲染]            │  正文 / 图文 / 链接 / 付费
├─────────────────────────────────────────┤
│ 👍 328   💬 56   ↗ 分享   ⋯ 更多         │  底栏：社交互动常驻
└─────────────────────────────────────────┘
```

### 类型
1. **纯文本/富文本** —— 折叠展开（>6 行）、作者置顶徽章
2. **图文/视频** —— 16:9 大图、视频时长角标、九宫格
3. **链接预览**（新）—— og:image + title + desc
4. **付费段落**（新）—— 试读 + 模糊遮罩 + 流内解锁

### 交互改进
| 改进点 | 旧 | 新 |
|---|---|---|
| 点赞 | 长按→弹窗→5 emoji | 底栏常驻+双击大拇指动画 |
| 阅读量 | 无 | 顶栏 👁 计数 |
| 折叠长文 | 无 | >6 行折叠 |
| 分享 | 顶部菜单 | 底栏常驻 |
| 置顶 | 不明显 | 左上 📌 徽章 |

## 四、付费/订阅流程新交互

对标公众号付费阅读 + 知识星球会员：
- **试读 + 流内解锁**：付费消息卡内嵌解锁入口，不跳页
- **价格档位 sheet**：月/季/年档位卡片，试用期提示
- **订阅状态卡**：到期时间、续费、退款入口

## 五、创建/编辑表单优化

- 分步表单（基础信息 → 分类标签 → 价格权限）
- 实时验证 + 进度指示
- 封面上传预览
